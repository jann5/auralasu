# Auralasu

Website and interactive configurator for handcrafted kitchen and furniture fronts.

**Live site:** [auralasu.pl](https://auralasu.pl)

## What this project showcases

- a polished public-facing website for a custom furniture brand;
- an interactive product/configuration experience;
- motion and 3D elements used to explain materials and options;
- responsive layouts for desktop and mobile;
- content and image handling prepared for a real business website.

## Stack

- Next.js 16
- React 19 and TypeScript
- Tailwind CSS
- Framer Motion
- React Three Fiber and Drei
- Sanity CMS
- Zustand

## Run locally

Requires Node.js 20 or newer.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm run start
```

## Project goal

Auralasu is a real product-oriented web project: the focus is not only on a visual landing page, but on helping a customer understand available fronts and move toward a configuration or enquiry.
